## **About the Project**  
This project is designed for analyzing microphone recordings, visualizing spectrograms, and calibrating multi-channel sound data. It includes modules for recording, calibration, and visualization.

---

## **Features**  
- Easily generate metadata files for Acoustic recordings with multiple microphones and recording arangements  
- Generate microphone calibration information
- Process data to generate a spectragram with option to calibrate
- Process data using metadata loaded from formatted json files

## **Getting Started**

### **Prerequisites**
- Python >= 3.x  
- Dependencies listed in `requirements.txt`  

### **Installation**

1. **Clone the repository:**  
    ```bash
    git clone 
    cd your-repo
    ```

2. **Create and activate a virtual environment:**  
    ```bash
    python -m venv venv
    .\venv\Scripts\activate
    ```

3. **Install dependencies:**  
    ```bash
    pip install -r requirements.txt
    ```
4. **Run Application:**
   streamlit run DroneAudioApp.py
